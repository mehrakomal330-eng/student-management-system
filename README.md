# 🎓 Student Management System

## 📌 Project Description

The **Student Management System** is a web-based application that allows users to manage student records efficiently. It performs all basic **CRUD operations** — Create, Read, Update, and Delete.

This system helps in storing, viewing, updating, and deleting student information in an organized way.

---

## 🚀 Features

* ➕ Add New Student
* 👀 View All Students
* ✏️ Update Student Data
* ❌ Delete Student
* 📋 User-friendly Interface
* 💻 Responsive Design
